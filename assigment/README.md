# Sound Euphonium Fan Site

A high-quality fan website for the anime _Sound Euphonium_ (Hibike! Euphonium).

## Features

- **Home Page**: Introduction with explore button.
- **Features**: Information about voice actors (Seiyuu).
- **Products**: Links to official merchandise on Shopee.
- **Service**: Anime synopsis and trailer.
- **Banner**: Image gallery of characters and scenes.
- **Sign Up**: User registration form with validation.
- **Navigation**: Consistent navigation across all pages.

## Technologies Used

- HTML5
- CSS3 (Responsive design with Flexbox)
- JavaScript (Form validation)
- Font Awesome (Icons)

## Structure

- `index.html`: Home page
- `feature.html`: Seiyuu information
- `product.html`: Merchandise
- `service.html`: Anime details and trailer
- `banner.html`: Image gallery
- `sign_up.html`: Registration form
- `navigation.html`: Information page
- `style.css`: Stylesheet
- `img/`: Images
- `img2/`: Additional images

## How to Run

Open `index.html` in a web browser.
